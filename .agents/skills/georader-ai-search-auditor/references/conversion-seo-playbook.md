# Conversion SEO Playbook

Use this reference when organic/AI search visibility must turn into consultation, diagnosis, report purchase, or monthly improvement revenue.

## Evidence Label

Conversion recommendations are usually `INDUSTRY_PRACTICE` unless backed by supplied analytics, experiments, or user research. GEORADER funnel packaging is usually `GEORADER_ASSUMPTION`.

## Review Areas

| Area | What To Check |
|---|---|
| CTA clarity | The next action says what the user gets, not just "submit" or "learn more" |
| Proof | The page/report includes real evidence, sample report, methodology, customer proof, or example outputs |
| Objection handling | Pricing, scope, timeline, accuracy limits, privacy, and deliverables are addressed |
| Trust | Forms and reports do not imply unsupported delivery, payment, diagnosis, or automation |
| Message match | Query intent matches page promise and CTA |
| Internal sharing | Report/page can help a marketer explain the issue to executives or agencies |
| Friction | Form fields, steps, and required details match the value of the offer |

## GEORADER Funnel Ideas

Possible offers:

- free mini AI search check.
- sample report.
- paid detailed diagnosis.
- consultation.
- monthly monitoring and improvement roadmap.
- agency/reseller support package.

Label these as `GEORADER_ASSUMPTION` unless the product/repository confirms them.

## Output Pattern

| Funnel step | Risk/opportunity | Evidence label | Recommendation | Safer wording |
|---|---|---|---|---|

## Trust Rule

If the form or CTA implies a real submission, report delivery, or diagnosis result, require backend/runtime evidence. If not verified, classify as P0 and recommend preview/provisional wording.
