# External Eval Patterns For GEO Strategy

Use this file to adapt evaluation thinking into GEORADER strategy without importing external tools or scores.

## Borrowable Patterns

- query evaluation rubrics
- prompt set coverage matrices
- persona and buyer-stage coverage
- competitor gap framing
- citation opportunity ledgers
- source-to-claim support questions
- evidence ledger formatting
- regression-style report QA checks

## Query Coverage Matrix

Check whether strategy covers:

- branded queries
- non-branded discovery queries
- competitor comparison queries
- problem-aware queries
- feature and use-case queries
- industry/category queries
- objection and risk queries
- local or region-specific queries when relevant

## Evidence Ledger Fields

For strategic recommendations, prefer:

- query family
- sampled prompt
- observation status
- brand mention status
- competitor mention status
- cited domain
- source-to-claim status
- evidence freshness
- seed contamination risk
- next verification step

## Opportunity Framing

Frame gaps as:

- "citation opportunity"
- "query coverage gap"
- "source support gap"
- "content evidence gap"
- "technical readiness issue"

Do not frame gaps as guaranteed lost traffic, guaranteed AI exclusion, or confirmed ranking loss unless RECORA evidence supports that exact claim.

## Handoff Rule

Strategy can draft recommendations with evidence labels. Publication decisions and final client-facing safety checks must go to `recora-recommendation-quality-gate-auditor`.
