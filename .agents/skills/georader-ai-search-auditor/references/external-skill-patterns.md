# External Skill Patterns

Use this reference when borrowing ideas from adjacent local skills.

## Borrowed Patterns

From local SEO, AI SEO, schema, copywriting, sales enablement, marketing psychology, programmatic SEO, and GEORADER LP review skills, borrow only these patterns:

- Evidence and source discipline for official guidance, research, and assumptions.
- AI citation readiness: clear definitions, comparison tables, FAQs, citable proof, source-backed claims, and no invented results.
- Technical SEO review shape: crawlability, indexability, rendering, structured data, mobile, and accessibility/agent-readiness checks.
- Schema accuracy rule: structured data must match visible page content and should be validated before implementation.
- Content architecture: hub/spoke, comparison pages, glossary, FAQ, industry pages, and no thin template pages.
- Conversion copy: clarity, proof, objections, CTA specificity, and honest claims.
- Sales/report output: scannable recommendations tied to business impact and next steps.
- GEORADER LP reviewer boundary: review/planning by default, implementation only in a separate task.

## Coordination Rule

This skill remains the authority for GEORADER-specific GEO/SEO/AI search strategy and report quality. Adjacent skills may inspire patterns, but do not import their full behavior, scripts, dependencies, or implementation permissions.

## Handoff Language

When work should leave this skill, say:

- "This skill can provide the evidence-labeled strategy and implementation plan. Actual code changes should be handled as a separate implementation task."
- "This is outside GEORADER GEO/SEO/AI search strategy, so I will keep the response to boundary guidance."
- "I can review the flow safely without seeing secrets; use redacted values or mock data."

## Do Not Borrow

- Unsafe claims that a tactic guarantees ranking or AI citation.
- Unverified statistics or citations from other skill text.
- Executable scripts, API integrations, or credential-dependent workflows.
- Broad social media consulting or unrelated marketing behavior.
- Permission to modify GEORADER production files.
